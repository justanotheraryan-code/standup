// command reference
standup log Log today's standup. Three prompts: yesterday, today, blockers.
standup show Display today's entry. Add -w to show the full week as a table.
standup digest Generate a Claude AI weekly digest. Streams live. Prompt-cached.
standup history List all weeks with entry counts.
standup add <text> Add a new todo item manually.
standup todos List all todos — open, done, skipped — with inferred project labels.
standup done <id> Mark a todo as done.
standup skip <id> Skip / dismiss a todo.
standup suggest Predict tomorrow's probable todos from your standup history. Requires ≥3 entries.
standup insights Multi-week growth analysis: velocity, blockers, project momentum, work patterns.
